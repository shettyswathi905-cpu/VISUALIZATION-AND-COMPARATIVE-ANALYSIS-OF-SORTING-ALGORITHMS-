# Sorting Algorithm Visualization and Comparative Analysis

## Project Overview
This project is a web-based tool that visually demonstrates and compares 11 sorting algorithms. It helps users understand how sorting works step-by-step and compares their time and space efficiency.


## Features
- Visual representation of 11 sorting algorithms  
- Step-by-step animation of sorting process  
- Comparison of time and space complexity  
- Simple and interactive user interface  
- Helps in understanding algorithm performance  


## Concepts Used
- Sorting Algorithms  
- Time and Space Complexity  
- JavaScript DOM Manipulation  
- Algorithm Visualization  


## Tech Stack
- HTML  
- CSS  
- JavaScript  


## How It Works
- Select a sorting algorithm  
- Input or generate an array  
- Watch step-by-step visualization  
- Get sorted output with performance comparison  


## Learning Outcomes
- Understanding of sorting algorithms  
- Visualization of algorithm execution  
- Improved JavaScript skills  
- Knowledge of time and space complexity
